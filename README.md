本仓库是论文《Robust Multimodal Mineral Identification based on a Parameter-Efficient Method》（基于参数高效方法的鲁棒多模态矿物识别）的开源实现。
项目提出了一种融合尺度偏移特征适配（Scale and Shift Feature Adaptation, SSF）与动态多模态信息瓶颈（Dynamic Multimodal Information Bottleneck, DMIB）的多模态矿物识别框架，
解决了实际地质勘探场景中存在的「模态缺失」「模态数据错误」等问题，在保证识别精度的同时，仅带来 13.8% 的参数量增长，具有较强的实用性和可扩展性。
